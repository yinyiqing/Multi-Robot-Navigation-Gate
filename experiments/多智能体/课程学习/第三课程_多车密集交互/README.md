# 第三课程：多车密集交互

## 这门课解决什么

第三课程才是人工设计多车交错、靠近、会车、目标聚集等密集交互场景。

它和第二课程不同：

- 第二课程：先把主线 A/D/D2 接回来，确认课程 actor 能进入 2车和 3车普通场景。
- 第三课程：在主线节点稳定后，再用手工密集 case 专门补多车交互短板。

## 后续入口

第三课程应等第二课程的 3A/3D/3D2 节点明确后再启动。

建议顺序：

1. 先用课程 3D2 best 做 300 集普通测试。
2. 如果普通测试稳定，再测三车轻密集诊断集。
3. 如果轻密集仍碰撞高，先做“两车冲突 + 第三车轻参与”的过渡集。
4. 不直接从当前状态硬训三车人工密集。
