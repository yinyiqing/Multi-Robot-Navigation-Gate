# 多智能体容量验证

本目录保存同一仿真环境下 2、3、5、10 车容量检查日志。

这些日志用于确认环境、reset、目标采样和 ROS/Gazebo 多车启动流程是否能承载不同机器人数量，不作为当前论文主线方法对照。
